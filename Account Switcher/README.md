#VeyraPersonalAddons – Account Switcher

A clean, lightweight Tampermonkey userscript that introduces a floating, draggable **Account Switcher** window to streamline profile management on DemonicScans.

## Features

- 🎛️ **Draggable UI Overlay:** Modern, compact layout modeled directly after the platform's native dashboard aesthetic.
- 🔒 **Local & Secure Storage:** Account credentials never touch the cloud; they are stored exclusively in plain text within your local browser extension manager context (`GM_setValue`).
- ⚡ **Background Automation Pipeline:** Switches sessions instantly utilizing a hidden background iframe framework to process asynchronous sequential sign-out and sign-in handlers securely.
- 🌊 **Seamless Experience:** Features a persistent blue loading veil that dynamically hides underlying web form mutations, refreshing your active tab only when the session update is complete.
